## Short Post

Built the missing Codex plugin GUI for Ollama this weekend.

Ollama already works with Codex, but most people do not know that, and using it means remembering a bunch of commands. This turns the official Codex App and CLI paths into plugin starter prompts and slash commands: status, setup, model switching, local/cloud models, restore, and safe dry runs.

No hand-editing Codex configs. It delegates to `ollama launch`.

https://github.com/daniel-p-green/ollama-codex-plugin
