Render the in-Codex Ollama control panel:

Use the `mcp__plugin_ollama-codex_ollama-codex__render_ollama_codex_panel` tool.

The panel renders inside Codex. It can check readiness, list local models, pull models, configure Codex CLI, restore Codex CLI, set up Codex App, switch Codex App models, and restore Codex App. App actions may restart Codex and require explicit confirmation in the panel.
