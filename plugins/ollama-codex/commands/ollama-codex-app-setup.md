---
description: Configure Codex App to use Ollama
allowed-tools: ["Bash(bash:*)"]
---

Configure Codex App with Ollama using the supported Ollama command:

```bash
bash ${CLAUDE_PLUGIN_ROOT}/scripts/ollama-codex.sh app-setup
```

Do not edit Codex App configuration files directly. Report that Codex App may restart as part of the setup flow.
