Configure Codex App with Ollama using the supported Ollama command:

```bash
bash ${CLAUDE_PLUGIN_ROOT}/scripts/ollama-codex.sh app-setup
```

Do not edit Codex App configuration files directly. If Ollama asks to restart Codex App, report the prompt and outcome accurately.
